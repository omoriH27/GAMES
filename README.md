### 1 数学基础GAMES001
[GAMES001-图形学中的数学](https://games-cn.org/games001/)主要总结归纳了其系列课程中会运用到的数学概念、理论和方法等，但总体上来说起到的是内容浅层引入和大纲性的作用，其中的每一部分实际上都可以作为一门完整的课程在课堂中进行讲授，具体内容如下：
- [ ] 线性代数基础
- [ ] 计算几何
- [ ] 旋转变换
- [ ] 主成分分析和奇异值分解
- [ ] 插值、拟合与采样
- [ ] 谱分析与傅里叶变换
- [ ] 概率论
- [ ] 场论初步
- [ ] 古典微分几何
- [ ] 微分方程
- [ ] 线性系统
- [ ] 最优化
- [ ] 机器学习
- [ ] 拓扑
